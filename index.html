Flashcards

FTP (File Transfer Protocol) Control

Transfers file data between a client and server (Control commands).

Port: 21

(Data connection often uses Port 20)

SSH (Secure Shell)

Provides secure remote login, file transfer (SFTP), and port forwarding.

Port: 22

Telnet

Provides unencrypted remote login and terminal access.

Port: 23

SMTP (Simple Mail Transfer Protocol)

Used for sending email messages between servers.

Port: 25

DNS (Domain Name System)

Translates human-readable domain names (like www.google.com) into IP addresses.

Port: 53 (UDP/TCP)

HTTP (Hypertext Transfer Protocol)

The foundation of data communication for the World Wide Web (unencrypted web traffic).

Port: 80

HTTPS (HTTP Secure)

Provides encrypted communication over HTTP, used for secure web traffic.

Port: 443
